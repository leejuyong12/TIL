# 관계 데이터 연산



관계 데이터 연산

- 관계 대수
  - 원하는 결과를 얻기 위해 데이터의 처리 과정을 순서대로 기술(절차 언어)
- 관계 해석
  - 원하는 결과를 얻기 위해 처리를 원하는 데이터가 무엇인지만 기술(비절차 언어)



데이터에 대한 처리 요구를 일반적으로 질의라고 한다.



관계 대수

- 관계 대수는 릴레이션을 연산한다.

- 일반 집합 연산자
  - 두 릴레이션의 차수가 같다. 즉, 두 릴레이션은 속성 개수가 같다.
  - 2개의 릴레이션에서 서로 대응되는 속성의 도메인이 같다. 단, 도메인이 같으면 속성의 이름은 달라도 된다.
